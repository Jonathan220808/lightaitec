# 粤剧咖啡 · 你是哪个角色 — Demo

这是 **第一杯咖啡** 的可玩 demo。零依赖、零安装。

## 怎么打开

### 方式一：双击（最简单）
直接双击 `index.html`，会在你默认的浏览器里打开。

### 方式二：用本地服务器（如果方式一有问题）
如果遇到 CORS 或脚本加载报错，用一个本地服务器：

```bash
# 进到 demo 目录
cd "E:/粤剧/demo"

# Python 3 自带（推荐）
python -m http.server 8000

# 或 Node 用户
npx serve .
```

然后打开浏览器访问 `http://localhost:8000`。

## 怎么玩

1. 进来后选 **中文 / EN**（右上角，可随时切）
2. 点 **开始**
3. 答 6 道生活化的问题（"咖啡像你哪一面"那种）
4. 等 2 秒"一台戏正在为你拉开"
5. 看你被匹配到哪个粤剧角色 + "情书式"的描写
6. 看一段 **演员独白占位**（实际版本需要你们的演员录制）
7. 收下一张 **带你角色名字的虚拟票**（可保存为 PNG 图片）

最佳体验：**用手机访问**（设计原本就是手机端）。桌面浏览器看是窄条形 — 这是预期的。

## 这个 Demo 的范围

✅ 完整 9 屏流程  
✅ 6 个真实粤剧角色（杜丽娘 / 李慧娘 / 红娘 / 花木兰 / 周世显 / 包拯）  
✅ 6 道生活化问题  
✅ 真实的性格匹配算法（余弦相似度）  
✅ 中英双语并行（每屏可切）  
✅ 虚拟票视觉、可保存为 PNG  

⛔ 演员独白用占位卡（需要真人录制）  
⛔ 角色卡里没有剧照（需要拍摄）  
⛔ 动效是基础版（没上 Framer Motion 那种高级动画）  
⛔ 角色只有 6 个（最终 30 个的清单要和粤剧顾问敲）  

## 文件结构

```
demo/
├── index.html      # 入口
├── styles.css      # 视觉设计（暗色 + 烫金 + 戏曲红）
├── data.js         # 角色 / 问题 / 中英文 i18n
├── app.js          # 状态机 + 匹配算法 + 票面生成
└── README.md       # 本文件
```

## 依赖

唯一外部依赖：保存票面成 PNG 时用了 [html2canvas](https://html2canvas.hertzen.com/)（CDN 加载）。其他全部本地。
